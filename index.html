Technical Audit
  • Dashboard source: public.dashboard_decision_watchlist_v.
  • Frontend cash-range selection is local only and does not mutate Supabase data.
  • Manual calculator uses controlled dropdowns to avoid false precision.
  • When DB values are not exactly on dropdown steps, calculator defaults to the nearest controlled input value.
  • Area is readonly and comes from the selected model row.
  • Purchase cost affects purchase fees and all-in cash need only; it does not change rent yield.
